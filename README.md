## Концепция

Аркадная казуальная игра, в которой клоун собирает предметы, брошенные зрителями на бортик циркового манежа.

Игрок управляет перемещением клоуна по арене. Цель — собрать как можно больше полезных предметов, избегая вредных, набрать максимальное количество очков и выполнить цель по количеству собранных предметов.
